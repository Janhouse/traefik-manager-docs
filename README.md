# Traefik Proxy Admin — Documentation

Documentation site for [Traefik Proxy Admin](https://github.com/Janhouse/traefik-proxy-admin),
a web panel for managing Traefik dynamic (and optionally static) configuration.

Built with [VitePress](https://vitepress.dev).

## Development

```bash
pnpm install      # install dependencies (Node 20+)
pnpm docs:dev     # start the local dev server
pnpm docs:build   # build the static site
pnpm docs:preview # preview the production build locally
```

## Deployment

The site auto-deploys to GitHub Pages via GitHub Actions on every push to `main`.
